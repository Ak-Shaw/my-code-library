Problem Description
Write an algorithm to find the Kth element in the array such that their only prime factors are 3,5 and 7. Note that 3,5, and 7 do not have to be factors, but it should not have any other prime factors.

Example: Such an array would contain 1,3,5,7,9,15,21,25,â€¦

We have to return the Kth element in the array.

Input format
First line contains an integer representing the value of K.

Output format
Print the Kth element in the array.

Constraints
1 <= K <= 1000

Sample Input 1
6

Sample Output 1
15

Explanation 1
The array will contain 1,3,5,7,9,15 after 6 operations.
